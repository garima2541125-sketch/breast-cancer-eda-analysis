Breast Cancer Data Analysis & Prediction
📌 Overview

This project focuses on exploratory data analysis (EDA) and predictive modeling using a breast cancer dataset. The objective is to analyze tumor characteristics, identify key features influencing diagnosis, and build a model to classify tumors as benign or malignant.

🎯 Objectives
Perform data cleaning and preprocessing
Conduct detailed exploratory data analysis
Identify important features affecting diagnosis
Build a machine learning model for prediction
📊 Exploratory Data Analysis
Analyzed feature distributions using histograms and boxplots
Compared tumor characteristics across diagnosis categories
Identified strong correlations among features (e.g., radius, perimeter, area)
Observed clear separation between benign and malignant tumors
🔍 Key Insights
Malignant tumors generally have higher radius, perimeter, and area
Strong correlation exists among size-related features
Dataset shows clear class separability, making it suitable for classification
Feature relationships indicate potential multicollinearity
🤖 Model Building
Model Used: Logistic Regression
Data Split: 80% training, 20% testing
Achieved approximately 93% accuracy
📈 Model Evaluation
High precision and recall values
Confusion matrix shows low misclassification
Model performs reliably in distinguishing tumor types
🛠 Tools & Technologies
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
📌 Conclusion
The dataset is clean and well-structured
Tumor size-related features are strong predictors of malignancy
EDA revealed clear patterns and relationships in the data
Logistic Regression model demonstrated strong predictive performance

👉 This project highlights how data analysis and machine learning can support healthcare decision-making, particularly in early cancer detection.
